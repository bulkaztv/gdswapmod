# GD Swap Mode

Mod do Geometry Dash 2.2 na Geode. Tryb swap - kontrola losowo zmienia się między dwoma graczami co 20-50 sekund.

## Jak działa
1. Jeden gracz klika **Hostuj** w menu SWAP
2. Drugi wpisuje IP hosta (np. z Radmin VPN) i klika **Dołącz**
3. Wchodzicie na dowolny level
4. Co 20-50 sekund losowo zmienia się kto gra — 3 sekundy przed zmianą pojawia się warning

## Budowanie
Mod buduje się automatycznie na GitHubie — po pushu pobierz plik `.geode` z zakładki Actions.
